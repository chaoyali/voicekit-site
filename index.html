FAQ

Common questions

Do I need any technical knowledge to set it up?

No. The stick arrives pre-flashed. Install Voice Cursor, pair the device, and you're done.

Does it work with any Mac app?

Yes — Voice Cursor works at the system level, so it works in any app: browsers, editors, email, Slack, everything.

What if I already own an M5Stick S3?

Flash it yourself using our open firmware. You'll still need a Voice Cursor subscription to use VoiceKit.

Is this only for macOS?

For now, yes. Windows and Linux support is on the roadmap.

What happens if I cancel my subscription?

You keep the hardware — the stick is yours. The software stops working once your subscription ends. If you cancel partway through the year, your refund is prorated for the months you've already used, minus a $20 hardware shipping fee. For example, after two months your refund covers the remaining ten months, less the $20 shipping.
